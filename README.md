# 🌱 Smart Agriculture Assistant Chatbot

An AI-powered chatbot developed using **Python**, **Streamlit**, and **Google Gemini API**. This chatbot is designed to assist farmers, agriculture students, and IoT enthusiasts by answering agriculture-related questions in a simple and interactive way.

---

## 📌 Project Overview

The Smart Agriculture Assistant is an intelligent chatbot that provides information related to:

- 🌾 Crop Management
- 💧 Irrigation Techniques
- 🌱 Soil Moisture
- 🌦️ Weather-related Farming Tips
- 🧪 Fertilizers and Nutrients
- 🐛 Pest and Disease Prevention
- 🏡 Greenhouse Farming
- 📡 IoT in Agriculture

The chatbot uses Google's **Gemini AI** model to generate accurate and natural language responses.

---

## 🎯 Objectives

- Develop an AI-powered agriculture assistant.
- Help farmers and students obtain farming information quickly.
- Demonstrate the integration of Artificial Intelligence with Agriculture.
- Build an interactive chatbot using modern AI technologies.

## 📦 Installation

### Step 1: Clone or Download the Project

```bash
git clone https://github.com/yourusername/Smart_Agriculture_Chatbot.git
```

Or simply download the project folder.

---

### Step 2: Install Python

Download Python from:

https://www.python.org/downloads/

---

### Step 3: Install Required Libraries

```bash
pip install -r requirements.txt
```

or

```bash
pip install streamlit google-generativeai python-dotenv
```

---

## 🔑 Getting a Gemini API Key

1. Visit https://aistudio.google.com/
2. Sign in with your Google account.
3. Click **Get API Key**.
4. Create a new API key.
5. Copy the generated key.

---

## ⚙️ Configure Environment Variables

Create a file named **.env**

Add your API key:

```text
GEMINI_API_KEY=YOUR_API_KEY_HERE
```

Example:

```text
GEMINI_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

---

## ▶️ Running the Project

Open the terminal inside the project folder.

Run:

```bash
streamlit run app.py
```

The application will automatically open in your default web browser.

## 💬 Sample Questions

You can ask questions like:

- What is soil moisture?
- What is drip irrigation?
- How can I improve crop yield?
- Which fertilizer is suitable for rice?
- What are greenhouse farming advantages?
- How does IoT help agriculture?
- How can farmers prevent crop diseases?
- Explain smart irrigation systems.
